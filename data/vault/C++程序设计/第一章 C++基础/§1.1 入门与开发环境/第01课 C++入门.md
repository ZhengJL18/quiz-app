## 第01课 C++入门

### 一、核心知识点

C++ 是 C 语言的扩展，增加了面向对象、泛型编程等特性。入门阶段需掌握：**程序基本结构**、**输入输出**、**变量与类型**、**常量与引用**。

#### 1. 程序基本结构
每个 C++ 程序至少包含一个 `main` 函数，执行入口在此。头文件用 `#include` 引入，如 `#include <iostream>` 提供输入输出功能。

```cpp
#include <iostream>
int main() {
    // 代码
    return 0;
}
```

#### 2. 输入输出
使用 `std::cout` 输出，`std::cin` 输入。需配合 `<<` 和 `>>` 运算符。

```cpp
int x;
std::cin >> x;           // 输入
std::cout << x << "\n";  // 输出并换行
```

#### 3. 变量与基本类型
变量声明格式：`类型 变量名;`。常用类型：`int`（整型）、`double`（双精度浮点）、`char`（字符）、`bool`（布尔）。

```cpp
int age = 20;
double pi = 3.14159;
char grade = 'A';
bool isPass = true;
```

#### 4. 常量与引用
- 常量：`const int MAX = 100;` 不可修改。
- 引用：`int &ref = age;` 别名，对 `ref` 操作即对 `age` 操作。

### 二、重点难点

- **`std::endl` vs `"\n"`**：`endl` 强制刷新缓冲区（影响性能），`"\n"` 仅换行。推荐用 `"\n"`。
- **引用 vs 指针**：引用必须初始化且不可更改指向，指针可空可改。引用更安全、简洁。
- **命名空间**：`using namespace std;` 可简化代码（如 `cout` 代替 `std::cout`），但大型项目慎用，易冲突。

### 三、简单示例

```cpp
#include <iostream>
int main() {
    int a, b;
    std::cout << "Enter two numbers: ";
    std::cin >> a >> b;
    int sum = a + b;
    std::cout << "Sum = " << sum << "\n";
    return 0;
}
```

#### 基础语法题
1. 以下哪个是合法的 C++ 标识符？  
   A. `2ndPlace`  B. `_value`  C. `int`  D. `my-var`
2. `const int x = 5; x = 10;` 会报错吗？为什么？

#### 简单编程题
3. 编写程序：输入一个整数，输出它的平方。  
4. 编写程序：交换两个整数的值（使用引用）。  

### 五、参考答案要点

1. **B**（标识符不能以数字开头、不能是关键字、不能含连字符）。
2. 会报错，`const` 变量不可修改。
3. 核心代码：`std::cin >> n; std::cout << n * n;`
4. 使用引用：`void swap(int &a, int &b) { int t = a; a = b; b = t; }`
