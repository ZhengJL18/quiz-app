## 第04课 函数基础

### 核心知识点

函数是 C++ 程序的基本组成单元，用于封装可复用的代码块。函数通过**参数**接收输入，通过**返回值**输出结果。函数定义包括返回类型、函数名、参数列表和函数体。

### 重点难点

1. **函数声明与定义**：声明告知编译器函数存在，定义提供实现。声明可多次，定义唯一。
2. **参数传递**：**值传递**复制实参，**引用传递**操作原变量。引用传递使用 `&` 符号。
3. **函数重载**：同名函数参数列表不同（类型或数量），编译器根据实参选择匹配版本。
4. **默认参数**：在函数声明中为参数指定默认值，默认参数必须从右向左连续定义。

### 示例

#### 示例1：值传递与引用传递
```cpp
#include <iostream>
void swapByValue(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}
void swapByRef(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}
int main() {
    int x = 5, y = 10;
    swapByValue(x, y);  // x,y 不变
    swapByRef(x, y);    // x=10, y=5
    return 0;
}
```

#### 示例2：函数重载
```cpp
#include <iostream>
int max(int a, int b) { return (a > b) ? a : b; }
double max(double a, double b) { return (a > b) ? a : b; }
int main() {
    std::cout << max(3, 7) << std::endl;      // 调用 int 版本
    std::cout << max(3.5, 2.1) << std::endl;  // 调用 double 版本
    return 0;
}
```

#### 示例3：默认参数
```cpp
#include <iostream>
void printInfo(int id, std::string name = "Unknown") {
    std::cout << "ID: " << id << ", Name: " << name << std::endl;
}
int main() {
    printInfo(101);           // 输出: ID: 101, Name: Unknown
    printInfo(102, "Alice");  // 输出: ID: 102, Name: Alice
    return 0;
}
```

3. 编写函数 `void increment(int &n)`，将引用参数加 1。在 `main` 中定义变量 `int x = 5`，调用函数后输出 `x` 的值。
