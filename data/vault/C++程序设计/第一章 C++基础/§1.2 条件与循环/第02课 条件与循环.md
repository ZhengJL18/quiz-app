# 第02课 条件与循环

## 一、核心知识点

**条件语句**控制程序分支，**循环语句**控制重复执行。两者共同构成结构化编程的基本控制流。

### 1. 条件语句

- **`if` 语句**：根据条件表达式（结果为 `bool` 类型）决定是否执行代码块。
  ```cpp
  if (x > 0) {
      cout << "positive";
  } else if (x == 0) {
      cout << "zero";
  } else {
      cout << "negative";
  }
  ```
- **`switch` 语句**：用于多分支选择，表达式必须是整型或枚举类型。每个 `case` 后需 `break` 防止贯穿。
  ```cpp
  switch (grade) {
      case 'A': cout << "excellent"; break;
      case 'B': cout << "good"; break;
      default: cout << "unknown";
  }
  ```

### 2. 循环语句

- **`for` 循环**：适用于已知循环次数。结构：`for (初始化; 条件; 更新)`。
  ```cpp
  for (int i = 0; i < 10; ++i) {
      sum += i;
  }
  ```
- **`while` 循环**：先判条件，后执行，适合未知次数。
  ```cpp
  int n = 10;
  while (n > 0) {
      cout << n--;
  }
  ```
- **`do-while` 循环**：先执行一次，再判条件，至少执行一次。
  ```cpp
  int x = 5;
  do {
      cout << x;
  } while (x > 10);  // 输出一次 5
  ```

### 3. 循环控制

- **`break`**：立即退出当前循环。
- **`continue`**：跳过本次循环剩余语句，进入下一次迭代。

## 二、重点难点

1. **条件表达式类型**：C++ 中非零值视作 `true`，零视作 `false`，但推荐显式比较。
2. **`switch` 贯穿（fall-through）**：忘记 `break` 会导致后续 `case` 无条件执行，常为 bug。
3. **循环变量作用域**：`for` 循环内定义的变量（如 `int i`）仅在循环体内有效。
4. **无限循环**：条件始终为真时发生，应确保循环变量更新正确或设置退出条件。

## 三、简单示例

### 示例1：判断闰年（条件嵌套）

```cpp
#include <iostream>
using namespace std;

int main() {
    int year = 2024;
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
        cout << year << " 是闰年";
    } else {
        cout << year << " 不是闰年";
    }
    return 0;
}
```

### 示例2：累加1到100（for循环）

```cpp
int sum = 0;
for (int i = 1; i <= 100; ++i) {
    sum += i;
}
cout << "1+2+...+100 = " << sum;  // 输出 5050
```

### 示例3：输入密码直到正确（while循环 + break）

```cpp
int password;
while (true) {
    cout << "请输入密码: ";
    cin >> password;
    if (password == 123456) {
        cout << "登录成功";
        break;
    }
    cout << "密码错误，请重试\n";
}
```
